# Agent Skills

Reusable skill definitions for AI agents (Enchanter, Hermes, etc.). Framework-agnostic, well-documented reference material.

## Skills

### web-development

Comprehensive reference for building modern, high-quality websites. Grounded in 2024–2026 web standards, accessibility laws, performance budgets, and contemporary design practice.

**Topics covered:**
- Color Systems (OKLCH, contrast, palette generation)
- Layout Systems (Grid, Flexbox, container queries, fluid typography)
- Typography (font selection, loading, type scales)
- Accessibility (WCAG 2.2, semantic HTML, focus management)
- Performance (Core Web Vitals, image optimization, JS budgets)
- SEO & Meta Tags (OG, Twitter cards, JSON-LD, sitemaps)
- Component Patterns (nav, demos, blog posts)
- Dark Mode
- Deployment checklists

**References:**
- `oklch-color-reference.md` — OKLCH color space guide
- `css-layout-patterns.md` — Common layout patterns and anti-patterns
- `web-performance.md` — Core Web Vitals, image optimization, caching
- `accessibility-reference.md` — WCAG 2.2, ARIA, keyboard accessibility
- `seo-meta-tags.md` — Meta tags, JSON-LD, technical SEO
